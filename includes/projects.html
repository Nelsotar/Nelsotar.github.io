Maze Solving Robot

A maze-solving robot made with JavaFX and becker.robots. Originally it was a robot that simply crawled through the entire maze until it found the correct location. I then modified it so there were two robots: an invisible first robot that acted as a “probe” where it explored the maze and created a list of directions to get to the destination directly, and a secondary visible robot that followed the directions collected by the probe.

I then created a GUI that a user could use to input maze size and the coordinates they would like the robot to navigate to. The GUI and the maze/robot are run in separate threads so they can run concurrently.

Made with: JavaFX, becker.robots (by Byron Weber Becker)

The n Queens Problem

A program/GUI I made for solving the n Queens problem and displaying its solutions. I originally created this to solve the problem with just 8 queens on one 8 x 8 board, but then I decided I wanted to expand it to include different board sizes. As of now it includes boards from 4 x 4 to 12 x 12 with queens to match. The positions included are all possible positions – not just fundamental positions.

Made with: JavaFX, CSS

Text-Based Uno Game

A text-based Uno game made for a class. This game has the ability to have from 2 to 10 players. Players may be computer or user-controlled, but the first player created must be a user. Includes functionality for all special cards such as: skip, reverse, wild, and wild draw 4.

Made with: Java
